
# Python to-do-app

#### Purpose and details

This is a Python app that serves as a simple to-do-list.

1. One can add entries via the UI
2. One can edit entries 


Backend  : Sqlite3 ( Reason for choosing : none of that initiate a server first bullshit )

Frontend : PyQT4
